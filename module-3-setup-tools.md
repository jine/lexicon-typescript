# Module 3: Setup and Tools

## Lesson 1: Installing Node.js and Package Managers

Node.js runs JS/TS outside browsers. npm/yarn manage dependencies.

### Setup Steps

1. Download Node.js from nodejs.org.
2. Verify: `node --version` and `npm --version`.
3. Initialize project: `npm init -y` or `yarn init`.

### Code Snippets

```bash
# Install TypeScript globally
npm install -g typescript

# Create tsconfig.json
tsc --init
```
